Prefetching content

Prefetching is an excellent way to improve perceived performance. I wrote a blog post explaining ways you can take advantage of this to make your page feel instant. If you're dealing with a complex database-driven application and you're not using any type of web caching, keep this option disabled. It will likely destroy your app server since it will increase the number of request each user makes to the server exponentially.
